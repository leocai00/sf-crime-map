Smart Route Planner

Crime Map

University District, Seattle

Stage One

In stage one, I build a crime map as a prototype using a set of sample geolocation data. The map indicates the criminal incidents happened around the University District, Seattle. Ideally, it would connect to the database of Seattle Police Department and gain instant updates. Users can use this map to get some sense of their surrounding environment and optimize their routes.

Next Step

My next step is to link the Safe Route Planner web application to a real crime database and build a responsive crime map. Besides that, I would like to analyze the crime data by its data attributes and generate a safety factor for each area in Seattle. Finally, basic on the safety factor of areas and some other repsonsive variables, the web app can provide commendation for route planning.
